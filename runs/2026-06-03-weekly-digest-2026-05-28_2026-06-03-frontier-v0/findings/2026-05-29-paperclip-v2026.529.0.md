# Paperclip v2026.529.0 - Inline annotations, skills CLI, sidebar controls, self-hosted admin flow, and live model discovery

## What Changed

Release v2026.529.0 introduced five major capabilities: inline document annotations with revision-aware threads and stable snapshots; skills as first-class resources with CLI commands (install, reset, audit, export, assign); user-scoped sidebar visibility controls; one-time admin claim flow for unclaimed self-hosted deployments; and live Claude model catalog refresh in the Claude Local adapter. Also includes workspace lifecycle enhancements and fixes for plan decomposition duplicate subtask prevention.

## Operator Implication

Operators managing unclaimed self-hosted deployments gain a streamlined bootstrap path via the new admin claim flow, reducing operational overhead. Skills management becomes CLI-first, improving programmatic governance and provenance tracking. Sidebar customization per user reduces UI clutter but maintains resource accessibility. Document annotation threads enable collaborative issue discussion with proper change tracking.
