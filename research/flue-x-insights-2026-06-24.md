# Flue — X Intelligence Digest (2026-06-24)

**Project:** flue (withastro/flue)

## Positioning
Flue is the Astro team’s TypeScript agent harness framework (“Astro for agents”). It emphasizes portability, lightweight sandboxes, and the “agent = directory + markdown skills” pattern.

Key primitives in 1.0 Beta (mid-June 2026): Workflows, Agents, Channels.

## Current Discussion
Active comparison with Vercel’s Eve framework:
- Flue strengths: portability, easy skill/subagent sharing, flexible sandboxing, true autonomy.
- Eve strengths: strong conventions, production plumbing, human-in-the-loop controls.

The debate is healthy ecosystem discussion around “deploy gravity vs portability.”

**Recommendation:** The harness vs platform debate is a strong signal for the “ecosystem” category.
