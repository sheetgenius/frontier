# Bitter Frontier

A weekly, receipted brief on the agentic-coding frontier — what changed this
week across the major coding agents and harnesses, what it costs the people who
run them, and what is worth trying or watching. Every claim carries its source.

Made by Bitter, which builds infrastructure for running AI agents in production.
We keep our own product out of the analysis.

---

_Setting up. This repository is becoming the open, contributable home of Bitter
Frontier: the published digests and profiles, the static site that renders them,
the source contracts, and the guidelines for contributing corrections. The live
publication is at https://frontier.bitter.sh._
