# Gemini CLI Source Notes

Watch Gemini CLI as a high-velocity open-source terminal agent and Google
ecosystem entrypoint.

High-signal areas:

- release-channel behavior: stable, preview, and nightly
- checkpointing, resume, context files, and `GEMINI.md`
- built-in tools, shell execution, web fetch/search, and output formats
- MCP, extensions, IDE integration, and GitHub Actions
- sandboxing, trusted folders, permissions, authentication, and enterprise use
- security advisories and regressions that affect trust in agent execution

Do not over-weight:

- nightly churn unless it indicates a likely upcoming stable change
- community complaints without linked issues, releases, or reproducible probes
- broad Gemini model news unless it changes CLI behavior or operator posture
- third-party security writeups without checking the official advisory or fix
