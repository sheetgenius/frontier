# Hermes Agent Source Notes

Watch Hermes as a broad self-improving agent platform.

High-signal areas:

- memory and skill creation
- skill self-improvement and Skills Hub compatibility
- subagents, delegation, toolsets, and RPC-like execution patterns
- scheduled automations and messaging gateways
- local, Docker, SSH, Singularity, Modal, Daytona, and other terminal backends
- sandboxing, command approval, security, and authorization
- context files, `SOUL.md`, `llms.txt`, and documentation for agents
- trajectory generation, research workflows, and training loops

Do not over-weight:

- generic "agent that grows with you" positioning without a concrete changed surface
- social hype without a primary repo, release, or docs link
- benchmark or usage claims without method and reproducible context
- features that are impressive but do not affect operator leverage, governance, receipts, or adapter posture
