# Pi Coding Agent Source Notes

Watch Pi as a deliberately minimal and highly extensible terminal coding
harness.

High-signal areas:

- extensions, skills, prompt templates, themes, and package distribution
- session tree, branching, sharing, export, and replay-like surfaces
- `AGENTS.md`, `SYSTEM.md`, compaction, dynamic context, and context control
- print/JSON mode, RPC, SDK, and embedding surfaces
- provider login, model switching, and subscription/API-key support
- design-principle changes about what the core refuses to own
- security and permission examples, especially if they stay extension-level

Do not over-weight:

- package demos that do not affect the core contract or operator workflow
- community packages without maintainer signal or usage evidence
- features Pi explicitly delegates to extensions unless that delegation exposes a Bitter opening
- generic comparisons to Claude Code/Codex without a concrete source-backed change
